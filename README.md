# KashViT

This is an OCR model for training and testing a Kashmiri OCR model.

## Overview

KashViT is designed to recognize and extract text from images containing Kashmiri script. This project provides tools for both training custom OCR models and testing their performance on Kashmiri text data.

## Project Structure

- `main.py` - Main entry point for training and testing the Kashmiri OCR model
- `test.py` - Test suite for validating model functionality
- `data/` - Directory for storing training and testing datasets
- `model_checkpoints/` - Directory for saving trained model checkpoints

## Getting Started

Run the main script to train or test the OCR model:
```bash
python main.py
```

Run tests:
```bash
python test.py
```
